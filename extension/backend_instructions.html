Socio.io Backend Instructions

The Socio.io extension requires a backend server to be running for content moderation to work. Follow these instructions to start the backend server.

Note: You need to start the backend server each time you want to use the extension. The extension will automatically connect to the backend once it's running.

Starting the Backend Server

Step 1: Locate the start_backend.bat file

Find the start_backend.bat file in the main Socio.io directory.

Step 2: Run the batch file

Double-click on start_backend.bat to run it. A command prompt window will open.

Important: Keep this window open while using the extension. Closing it will stop the backend server.

Step 3: Verify the backend is running

You should see messages in the command prompt window indicating that the Flask server has started.

The extension will automatically detect the running backend and start working.

Troubleshooting: If you see any error messages in the command prompt window, make sure Python is installed and all required packages are installed. You can install the required packages by running:

pip install flask cryptography

Using the Extension

Once the backend is running:

  1. The extension icon should show "ON" instead of "ERROR"
  2. Browse to any website and the extension will automatically filter content
  3. You can view filtering statistics by clicking on the extension icon

If you have any questions or issues, please refer to the README file or contact support.
